Pour lancer le jeu, ouvrez simplement le fichier index.html.

Ce jeu est le deuxième opus d'une trilogie (seul le menu n'est pas complètement terminé, mais cela n'aura pas d'impact sur le jeu !).

Le projet vient d'un gars (moi) qui s'est demandé un soir : "Est-ce qu'un projet réalisé uniquement avec des prompts est possible ?". Alors, en 2 semaines, j'ai mené à bien ce projet.
